Practicing C# by doing the W3 Schools Tutorial (Found here: https://www.w3schools.com/cs/cs_getstarted.php)

Developing on Lunix (Ubuntu Distro). Environment setup instructions can be found here: https://learn.microsoft.com/en-us/dotnet/core/install/linux.

Creating a program(Ubuntu) in CMD:
* Install dotnet
* Check that dotnet in installed: `dotnet --version`
* Create project: `dotnet new console -n <Project Name>`
* Go into the <Project Name> directory.
* Run the program: `dotnet run`
